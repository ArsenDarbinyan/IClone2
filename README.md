# iClone.evn - Առցանց Խանութ

Պրեմիում դասի քոփիներ՝ հասանելի գներով։

## Նկարագրություն

iClone.evn-ը ժամանակակից առցանց խանութ է, որը նախատեսված է էլեկտրոնիկայի վաճառքի համար։ Նախագիծը ներառում է ինչպես հաճախորդների միջերես, այնպես էլ ադմինիստրատիվ վահանակ՝ ապրանքների կառավարման համար։

## Ֆունկցիոնալություն

### Հաճախորդների համար
- 📱 Ապրանքների կատալոգ
- 🔍 Որոնում և ֆիլտրացիա
- 🛒 Զամբյուղ
- 📞 WhatsApp-ով պատվերներ
- ⭐ Հաճախորդների կարծիքներ
- 📱 Responsive դիզայն

### Ադմինիստրատորի համար
- 🔐 Պաշտպանված մուտք
- ➕ Ապրանքների ավելացում
- ✏️ Ապրանքների խմբագրում
- 🗑️ Ապրանքների ջնջում
- 📊 Ապրանքների կառավարում

## Տեխնոլոգիաներ

### Frontend
- HTML5, CSS3, JavaScript (ES6+)
- Font Awesome պատկերակներ
- Responsive դիզայն

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Տեղադրում և գործարկում

### Պահանջներ
- Node.js (v14 կամ բարձր)
- MongoDB (v4.4 կամ բարձր)
- npm կամ yarn

### Քայլեր

1. **Ներբեռնեք պրոյեկտը**
   ```bash
   git clone <repository-url>
   cd iclone-evn
   ```

2. **Տեղադրեք կախումները**
   ```bash
   npm install
   ```

3. **Կազմաձևեք շրջակա միջավայրը**
   ```bash
   cp .env.example .env
   ```
   Խմբագրեք `.env` ֆայլը՝ ձեր MongoDB կապակցությամբ։

4. **Սկզբնավորեք տվյալների բազան**
   ```bash
   node init-db.js
   ```

5. **Գործարկեք սերվերը**
   ```bash
   # Մշակման ռեժիմում
   npm run dev
   
   # Արտադրության ռեժիմում
   npm start
   ```

6. **Բացեք բրաուզերը**
   - Գլխավոր էջ: http://localhost:3001
   - Ադմին վահանակ: http://localhost:3001/admin

## Ադմինիստրատիվ մուտք

- **Օգտանուն**: `admin`
- **Գաղտնաբառ**: `admin123`

> ⚠️ **Կարևոր**: Արտադրության միջավայրում փոխեք լռելյայն գաղտնաբառը։

## API Էնդփոյնթներ

### Ապրանքներ
- `GET /api/products` - Ստանալ բոլոր ապրանքները
- `GET /api/products/:id` - Ստանալ կոնկրետ ապրանք
- `POST /api/products` - Ավելացնել նոր ապրանք
- `PUT /api/products/:id` - Թարմացնել ապրանք
- `DELETE /api/products/:id` - Ջնջել ապրանք

### Ադմինիստրացիա
- `POST /api/admin/login` - Մուտք ադմինիստրատիվ վահանակ
